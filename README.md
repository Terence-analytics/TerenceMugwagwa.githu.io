# Terence Analytics Portfolio

This repository hosts the GitHub Pages website for **Terence Analytics**, a data analytics portfolio showcasing work across Excel, Power BI, Tableau, Python, SQL, and machine learning.

The site is a static HTML/CSS/JavaScript portfolio based on the Massively template by HTML5 UP. It is intended to deploy directly from the repository root through GitHub Pages.

## Structure

- `index.html` - main portfolio page and project gallery
- `assets/` - CSS, JavaScript, Sass, and webfont files
- `images/` - portfolio imagery and page assets
- `generic.html` and `elements.html` - original template reference pages

## Project Links

Portfolio cards link to related project folders in [`Terence-analytics/My_Projects`](https://github.com/Terence-analytics/My_Projects).

## Deployment

GitHub Pages can serve this site directly from the repository root. Keep `index.html`, `assets/`, and `images/` at the root level so relative file paths continue to work.
